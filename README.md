# 🎪 BLADE CIRCUS

> ワンタップで楽しむ 3D ナイフ投げカーニバルゲーム

回転するターゲットホイールにブレードを突き刺していく一発操作のアクションゲームです。すでに刺さっているブレードに当てないように投げ、リンゴを割ってコンボを稼ぎ、より速く・より難しいホイールをクリアしていきます。5 ステージごとに鋼鉄製のボスホイールが登場します。

---

## ✨ 特徴
- **ワンタップ操作** — タップまたはクリックだけでプレイできるシンプルな操作性
- **回転ホイール** — 既に刺さったブレードに当てないように狙うタイミングゲーム
- **リンゴでコンボ** — ホイール上のリンゴを割ってコンボを獲得
- **ステージ進行** — 進むほどホイールは速く、トリッキーに
- **ボスホイール** — 5 ステージごとに鋼鉄製のボスホイールが出現
